# API描述
查看监控项最新数据，v0.1.6开始加入。

## API地址
~~~
/api/v1/agent/${agentId}/app/${appId}/item/${itemId}/latest
~~~
其中：
* `${agentId}` - Agent ID
* `${appId}` - App ID
* `${itemId}` - 监控项ID

## 请求方法
~~~
GET
~~~

## 请求参数
无
